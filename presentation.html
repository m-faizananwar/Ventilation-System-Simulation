  • Fire Hazards ‚Äî Smoke infiltration requires immediate mitigation

üß™ Testing & Validation Strategy

Unit Testing

  • Masking Logic ‚Äî Verify AND operations correctly isolate bits 0, 1-7, and 0-7
  • Branch Logic ‚Äî Inject values 99 (Safe) and 101 (Danger) to test conditionals
  • LED Drawing ‚Äî Confirm pixel colors at expected memory addresses

Integration Testing

  • Smoke Switch Test ‚Äî Toggle switch, LED must turn red within 1 cycle
  • Noisy Data Test ‚Äî Random bits in unused regions should be ignored
  • MQTT Round-Trip ‚Äî Verify command delivery latency < 500ms
